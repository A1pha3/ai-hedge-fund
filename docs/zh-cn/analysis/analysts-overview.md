# AI 分析师介绍

本系统集成了 16 位不同投资风格的 AI 分析师，每位分析师都基于真实投资大师的策略和理念。

## 价值投资派

### 1. Ben Graham (本·格雷厄姆)
- **称号**: 价值投资之父
- **策略**: 寻找具有安全边际的隐藏宝石
- **关注点**: 低估值、财务稳健、安全边际
- **经典理论**: 《证券分析》、《聪明的投资者》

### 2. Charlie Munger (查理·芒格)
- **称号**: 巴菲特的合伙人
- **策略**: 只买优秀企业，以合理价格买入
- **关注点**: 企业质量、长期价值、能力圈
- **经典理论**: 多元思维模型、能力圈原则

### 3. Warren Buffett (沃伦·巴菲特)
- **称号**: 奥马哈先知
- **策略**: 寻找 wonderful companies at fair prices
- **关注点**: 护城河、管理层质量、长期竞争优势
- **经典理论**: 价值投资、护城河理论

### 4. Michael Burry (迈克尔·伯里)
- **称号**: 大空头
- **策略**: 逆向投资者，寻找深度价值
- **关注点**: 被低估的资产、市场错误定价
- **经典案例**: 2008年次贷危机做空

### 5. Mohnish Pabrai (莫尼什·帕伯莱)
- **称号**: Dhandho 投资者
- **策略**: 低风险、高回报的投资机会
- **关注点**: 低风险赌注、高概率成功
- **经典理论**: Dhandho 投资框架

## 成长投资派

### 6. Cathie Wood (凯茜·伍德)
- **称号**: 成长股女王
- **策略**: 相信创新和颠覆的力量
- **关注点**: 技术创新、颠覆性公司、未来趋势
- **代表**: ARK Invest 创始人

### 7. Phil Fisher (菲尔·费舍)
- **称号**: 成长股之父
- **策略**: 深入研究、"闲聊"调研
- **关注点**: 公司质量、增长潜力、管理层
- **经典理论**: 《怎样选择成长股》

### 8. Peter Lynch (彼得·林奇)
- **称号**: 实用主义投资者
- **策略**: 寻找日常生活中的"十倍股"
- **关注点**: 消费者体验、可理解的企业
- **经典理论**: "投资你所知道的"

### 9. Stanley Druckenmiller (斯坦利·德鲁肯米勒)
- **称号**: 宏观投资传奇
- **策略**: 寻找具有增长潜力的非对称机会
- **关注点**: 宏观趋势、非对称风险收益
- **经典成就**: 量子基金核心成员

## 专业分析派

### 10. Aswath Damodaran (阿斯沃斯·达莫达兰)
- **称号**: 估值之王
- **策略**: 基于故事、数据和纪律的估值
- **关注点**: 内在价值、估值模型、故事与数字
- **地位**: 纽约大学斯特恩商学院教授

### 11. Bill Ackman (比尔·阿克曼)
- **称号**: 激进投资者
- **策略**: 采取大胆立场，推动变革
- **关注点**: 公司治理、价值释放、 activism
- **代表**: Pershing Square 创始人

### 12. Rakesh Jhunjhunwala (拉凯什·君君瓦拉)
- **称号**: 印度大牛市
- **策略**: 印度市场价值投资与成长结合
- **关注点**: 印度经济、消费增长、长期趋势
- **地位**: 印度沃伦·巴菲特

## 技术分析派

### 13. Valuation Agent (估值分析师)
- **功能**: 计算股票的内在价值
- **方法**: DCF、相对估值、交易信号生成
- **输出**: 买入/卖出信号、目标价格

### 14. Sentiment Agent (情绪分析师)
- **功能**: 分析市场情绪
- **方法**: 新闻情绪分析、社交媒体情绪
- **输出**: 情绪信号、信心指数

### 15. Fundamentals Agent (基本面分析师)
- **功能**: 分析基本面数据
- **方法**: 财务报表分析、比率分析
- **输出**: 基本面评分、健康度评估

### 16. Technicals Agent (技术分析师)
- **功能**: 分析技术指标
- **方法**: 图表模式、技术指标、趋势分析
- **输出**: 技术信号、支撑阻力位

## 风险与组合管理

### Risk Manager (风险管理师)
- **功能**: 计算风险指标，设置仓位限制
- **方法**: VaR、波动率分析、相关性分析
- **输出**: 风险评分、仓位建议

### Portfolio Manager (投资组合经理)
- **功能**: 做出最终交易决策
- **方法**: 综合所有分析师信号、风险管理
- **输出**: 最终交易决策、订单生成

## 分析师信号汇总

每位分析师都会对每只股票给出以下信号之一：

- **BULLISH (看涨)**: 建议买入
- **BEARISH (看跌)**: 建议卖出
- **NEUTRAL (中性)**: 建议持有或不操作

信号附带：
- **置信度 (Confidence)**: 0-100%，表示分析师对判断的信心
- **推理 (Reasoning)**: 详细的分析逻辑和理由
