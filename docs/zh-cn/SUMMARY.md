## 项目简介

本项目是一个概念验证（Proof of Concept，概念验证）系统，旨在探索使用大型语言模型（LLM，Large Language Model）进行投资决策的可能性。

### 核心特性

- **18 位投资大师模拟**：系统模拟了 18 位著名投资大师的分析风格，包括价值投资的巴菲特、成长投资的彼得·林奇、宏观策略的斯坦利·德鲁肯米勒等
- **多智能体协作**：通过多智能体协作的方式生成交易建议，从多个角度分析市场
- **教育价值优先**：核心目标是帮助学习者理解 AI 在金融领域的应用，而非实际投资

### 系统定位

> **⚠️ 重要声明**
>
> 本系统**仅用于教育目的**，不提供任何投资建议，也不应用于实际交易。所有决策仅供参考，不构成投资建议。

## 文档体系设计理念

> **为什么采用四级文档体系？**

我们采用了渐进式学习理论（Zone of Proximal Development，最近发展区）来设计文档体系：

**1. 降低认知负荷**
- 将复杂知识拆分为四个难度层级
- 每个层级都在前一层的知识基础上构建
- 避免信息过载，让学习者在舒适的节奏下进步

**2. 建立完整认知框架**
- Level 1（入门）：建立信心，消除恐惧
- Level 2（核心概念）：建立知识体系
- Level 3（进阶分析）：培养深度思考能力
- Level 4（专家设计）：激发创新能力

**3. 满足不同目标的学习者**
- 用户路径：快速上手，实际使用
- 开发者路径：理解实现，二次开发
- 研究者路径：深入原理，探索创新

```
Q: 你的主要目标是什么？
├── 快速上手，使用系统进行分析
│   └── → 用户路径（Level 1 + Level 2 部分）
├── 参与开发，定制和扩展系统

└── 深入研究，理解 AI 投资决策原理
    └── → 研究者路径（Level 2 + Level 3 + Level 4）

├── 非技术背景，只关心使用

├── 有编程经验，想参与开发
│   └── → 开发者路径

    └── → 研究者路径
```

完成相应级别学习后，请使用以下清单进行自我检验。

### Level 1 自检清单

完成 Level 1 学习后，你应该能够：

- [ ] **环境配置**：能够独立安装 Poetry、配置 API Key
- [ ] **基本运行**：能够成功运行股票分析并查看结果
- [ ] **命令行使用**：能够使用常用命令行参数
- [ ] **Web 界面**：能够通过 Web 界面完成基本分析

### Level 2 自检清单

完成 Level 2 学习后，你应该能够：

- [ ] **智能体理解**：能够解释 18 个智能体各自的投资风格
- [ ] **工作流理解**：能够描述 LangGraph 工作流的执行流程
- [ ] **数据理解**：能够理解系统如何获取和处理金融数据
- [ ] **风险管理**：能够解读风险指标的含义
- [ ] **回测理解**：能够理解回测框架的工作原理

### Level 3 自检清单

完成 Level 3 学习后，你应该能够：

- [ ] **智能体开发**：能够开发自定义智能体
- [ ] **高级回测**：能够进行高级回测并解释结果
- [ ] **性能优化**：能够优化系统性能
- [ ] **问题诊断**：能够独立诊断和解决系统问题
- [ ] **生产部署**：能够将系统部署到生产环境

### Level 4 自检清单

完成 Level 4 学习后，你应该能够：

- [ ] **架构理解**：能够深入理解系统架构设计
- [ ] **源码理解**：能够阅读和理解核心源码
- [ ] **扩展设计**：能够设计合理的系统扩展方案
- [ ] **代码贡献**：能够成为项目的活跃贡献者
