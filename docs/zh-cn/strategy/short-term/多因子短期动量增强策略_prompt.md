### 一、引力量价复合因子

**数据来源：逐笔成交数据**

民生证券2025年1月发布的系列报告验证了两个核心逻辑：

**主买成交特异性因子**（主买资金的"特异性"流入）：
- 截面上，某只股票在当日全市场主买成交量中的占比突然显著扩大——代表主力关注
- 日内"主买时刻"出现的次数越多，表明需求持续，未来上涨概率越大
- 实证表现：Rank IC均值**11.05%**，多空年化收益**42.48%**，仅多头年化超额收益便达**9.05%**

**虹吸效应因子**（主力撤离后的反转逻辑）：
- 被主力卖压流出的股票，在极度超卖后出现资金回流与价格修复
- 对应前文"赚钱效应扩散"中龙头打开空间、低位补涨的逻辑
- 实证表现：Rank IC均值**8.31%**，多空年化收益**29.36%**，多头年化超额收益**8.70%**

> **核心合成方式**：将"主买成交特异性"与"虹吸效应"两个因子**等权合成**，形成"引力场"因子：
> - 月频回测：Rank IC均值**11.14%**，多头年化超额收益**10.69%**
> - 周频回测：多空年化收益达**66.28%**，多头年化超额收益跃升至**14.21%**，大盘、中盘、小盘股票均有效（多空收益分别为24%、35%、39%）

在你的综合模型框架中，该因子可与原有的**板块热度（权重分30%主力资金流向）**与**个股量价类（权重35%）**直接融合，替代原有的简单主力资金流入指标。

### 二、大单分钟主动资金流因子

**数据来源：逐笔成交的分钟聚合**

开源证券2026年2月最新研究证实，大单主动成交资金中蕴含着持续稳定的选股Alpha——而且**小单几乎无效**：

- **大单因子**：Rank IC均值**0.045**，Rank ICIR达**3.1**，信息比率**3.18**，选股表现稳健优异
- **小单因子**：Rank IC均值仅**-0.006**，信息比率仅**0.56**，完全没有选股效果，印证了"散户资金流噪声极高"这一量化核心结论

该研究还提出了三重精度增强方案：

1. **时段切割增强**：将交易时段区分为前半小时、第1小时～第4小时，其中**第1小时**构造的因子选股效果最优（Rank ICIR = 3.23），这表明每日开盘首个小时即奠定了一日资金的主导性
2. **涨跌幅切割增强**：仅选取1分钟涨跌幅较低（即波动平稳、噪音相对较小）的60%分钟，可使Rank IC均值从0.045提升至**0.053**（Rank ICIR = 3.34）
3. **情景合成增强**：将切割、时段、情景三种因子合为一体后，Rank ICIR最高可达**4.54**，信息比率达到**4.95**——这在A股量价类因子中属于顶级水平

### 三、短线攻击斜率因子

**数据来源：日线**

"攻击斜率"指价格变化的加速度，衡量上涨是正在加速还是减速。业界检验表明，该因子在趋势市（上涨或下跌）中表现极佳，在震荡市中效果锐减，因此天然适合配合"板块热度"使用——只在热点板块趋势明确时使用该因子。

这是当前公募与私募中高频量价策略最常使用的底层参数之一，原因在于它既能低成本获取（仅需日线收盘价），又能在趋势市场中快速捕捉高弹性候选标的。在板块盈利扩散初期，龙头股进入高位滞涨期，同板块低位股的短线攻击斜率一旦陡峭化，往往意味着补涨即将启动。

### 四、有限关注因子

**数据来源：日线**

BigQuant量化平台总结的研究表明，基于散户有限关注行为的综合因子，在小盘股中效果尤其显著，其本质在于通过量化散户"追涨杀跌"行为来获取超额收益。该因子特征包括：

- **综合构建逻辑**：由异常换手率 + 异常成交量 + 极端收益 + 龙虎榜指标**四位一体**合成，全面刻画散户情绪
- **小盘效应显著**：最适用场景是中小盘股票——在中证1000指数成分池内的选股效果最佳
- **指增场景数据**：将有限关注因子与中证1000指数增强策略叠加后，年化收益从纯中证1000指数的**5.3%大幅提升至18.9%**，增量高达**13.7个百分点**，夏普比从0.21提升至0.81

**对你的策略的点睛价值**：前文要求筛选5日涨幅20%的极端目标，本质上是在寻找短期内将受到市场高度关注、情绪快速升温的高弹性标的。有限关注因子正是这一机制最直接的量化表达，因此建议作为独立辅助因子置于板块热度与个股量价值之间，以强化动量与情绪共振效应。

### 五、涨停基因因子

**数据来源：日线**

近2～4个交易日内是否有过涨停，这是表征一只股票**股性活跃**的最直接指标：涨停本身在A股就是"股性鉴定器"，涨停过的股票短期内被再度买卖推升的活跃度显著高于非涨停股。

该因子在多因子模型中极其简单即可实现，且不会出现过度优化，是业界高频选股策略中广泛使用的标准风险调整参数，几乎没有完全剔除的。在许多动量策略回测中，该因子的信息比率均可超过2.0。

### 六、市场情绪择时开关（辅助级）⭐

**数据来源：日频+板块事件数据**

该因子并非选股因子，而是整个策略的**仓位调控开关**，直接决定风险敞口：

| 指标 | 解读与衡量标准 | 用途 |
| :--- | :--- | :--- |
| **连板晋级率** | 昨日涨停股次日的继续涨停比例 | 高于50%为极强情绪，低于20%为极度冷却 |
| **炸板率** | 盘中触板但未封死的比例 | 高于40%为情绪恶化信号，应降低仓位 |
| **全市场成交额** | 沪深两市合计金额 | 高于万亿为活跃，低于6500亿为极低流动性，需保守操作 |
| **昨日涨停指数** | 同花顺/通达信公式的行业标准指标 | 连续2日录得跌幅为假突破警示 |
| **赚钱效应扩散程度** | 高位涨停股与低位新启动个股比值 | 验证赚钱效应是否存在扩散行为 |

### 🧩 精简版多因子模型：用最少因子实现更强预测

基于以上六大因子提出优化版多因子模型，权重可根据市场数据进一步调试优化：

| 模型层 | 使用方法 | 权重 |
| :--- | :--- | :--- |
| **择时层：市场情绪开关**（前置仓位控制） | 连板晋级率+炸板率+成交额+Pilot扩散指标 | 情绪高涨时段允许100%目标仓位；情绪中性限制至60%；情绪冰点直接降至30%或空仓 |
| **个股层：核心α来源**（按实证多头超额能力定权重） | 引力量价复合因子**35%** + 大单主动资金流时段增强因子**25%** + 攻击斜率因子**15%** + 有限关注因子**15%** + 涨停基因因子**10%** | → 合成总预期得分 |

**应用流程**：
1. 早盘前：通过市场情绪开关决定**当日最大持仓总额**
2. 选中板块：仅在上一节所述的板块热度前5板块成分股内使用本模型
3. 在合格候选股池内，分别计算五种因子得分（各标准化为0～100，防止偏倚）
4. 按上述权重加权得到最终综合分，取前**3～5只**买入，单只仓位上限25%

该精简框架相比原始10+因子模型，**每个因子均有业界真凭实证支撑**，且高频低频数据互补（日频成本低 + 分钟/逐笔获取信息增量），有望在3-9日的时间窗口中保持高预测力。

# 额外增强建议
分档止盈：日内如果出现连续涨停或高斜率冲高，在第2~3天就触发动态止盈，而不是死等5日窗口的20%目标

窗口自适应：引入"市场状态探测器"——确定当前是趋势市还是震荡市（过去20个交易日沪深300指数波动区间除以其均值）。震荡市时缩小窗口至3日，趋势市时可适度放宽至3~9日

信号质量过滤：当情绪温度计处于中性或低迷区间时，仅保留信号最强的1~2只股票，其余不加仓，避免在低效窗口浪费资本
