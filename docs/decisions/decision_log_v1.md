## 决策清单

### 决策点 1：硬止损比例

| 项目 | 内容 |
|------|------|
| **默认值** | -7% |
| **决定** | ✅ 确认 -7% |
| **理由** | A 股日均波动 2-3%，-7% 约为 2-3 倍标准差，既能过滤日常噪音，又能在系统性下跌中及时止损。与 Level 2 波动止损（2×ATR）形成互补。 |

### 决策点 2：行业暴露上限

| 项目 | 内容 |
|------|------|
| **默认值** | 25%（申万一级，31 个行业） |
| **决定** | ✅ 确认 25% |
| **理由** | 等权基准约 3.2%/行业，25% 上限允许约 8 倍超配，在保持分散化与允许行业集中获取超额间取得平衡。HHI > 0.15 时附加约束生效。 |

### 决策点 3：涨幅榜策略

| 项目 | 内容 |
|------|------|
| **默认值** | 保留为第五策略，权重 10% |
| **决定** | ⏳ MVP 阶段暂不纳入，v1.5 考虑 |
| **理由** | MVP 聚焦四策略核心框架的完整性验证。涨幅榜策略本质是动量追涨，与策略 A（趋势）有信号重叠，过早纳入增加调参复杂度。现有 `batch_run_hedge_fund.py` 逻辑作为独立脚本保留。 |

### 决策点 4：做空信号

| 项目 | 内容 |
|------|------|
| **默认值** | 仅标记不执行 |
| **决定** | ✅ 确认 |
| **理由** | A 股融券门槛高（50 万起 + 券源紧张），且融券费率极不透明。做空信号保留为冲突仲裁的输入（安全优先规则），不生成实际做空单。 |

### 决策点 5：最大持仓天数

| 项目 | 内容 |
|------|------|
| **默认值** | 20 日（基本面驱动可延至 40 日） |
| **决定** | ✅ 确认 20/40 日双档 |
| **理由** | 20 日覆盖一个完整月度周期，适合技术/动量驱动。基本面驱动的持仓（入场主要信号来源为策略 C）允许延至 40 日，留出财报催化剂兑现时间。判定依据：入场时策略 C 贡献占比 > 50% 则为基本面驱动。 |

### 决策点 6：Sortino 目标

| 项目 | 内容 |
|------|------|
| **默认值** | > 1.0（首版） |
| **决定** | ✅ 确认 1.0 |
| **理由** | 首版目标设定保守合理。Sortino > 1.0 意味着下行风险调整后的收益为正。v2 追求 1.5 需经过至少 3 个月模拟盘验证。 |

### 决策点 7：信号有效期

| 项目 | 内容 |
|------|------|
| **默认值** | 2 个交易日 |
| **决定** | ✅ 确认 2 日 |
| **理由** | A 股 T+1 制度下，最快 T+1 执行，信号在 T+2 前有效。超过 2 日的信号需重新评估（衰减检查器负责）。 |

### 决策点 8：Layer B/C 权重

| 项目 | 内容 |
|------|------|
| **默认值** | B: 0.4 / C: 0.6 |
| **决定** | ✅ 确认 0.4/0.6，标记为**高敏感参数** |
| **理由** | Layer C 包含 18 个 Agent 的深度分析（含 LLM 推理），信息量显著大于 Layer B 的纯规则计算，给予 60% 权重合理。但此参数需在 Phase 5 的 walk-forward 回测中验证，若 Layer B 独立表现优于预期则考虑上调至 0.5/0.5。 |
| **验证计划** | Phase 7 参数敏感性分析中做 0.3~0.5 / 0.5~0.7 扰动 |

### 决策点 9：Agent 权重分配

| 项目 | 内容 |
|------|------|
| **默认值** | 投资大师 72%（12 × 6%）/ 技术分析师 28%（6 × 4.67%） |
| **决定** | ✅ 确认，标记为**高敏感参数** |
| **理由** | 投资大师 Agent 具备多维度分析能力（基本面 + 估值 + 风格），信息密度高于单一维度的技术分析师。72/28 分配反映了质量优先原则。 |
| **验证计划** | Phase 5 A/B 回测中作为对照变量之一，测试 60/40 和 80/20 两个替代方案 |

### 决策点 10：阈值再标定频率

| 项目 | 内容 |
|------|------|
| **默认值** | 月度，±0.05 |
| **决定** | ✅ 确认 |
| **理由** | 月度频率在反应市场结构变化与避免过度拟合间取平衡。±0.05 的单月限制防止阈值剧烈漂移。P3 优先级实现。 |

### 决策点 11：策略资金容量

| 项目 | 内容 |
|------|------|
| **默认值** | 1-5 亿 AUM 最优 |
| **决定** | ✅ 确认 |
| **理由** | 流动性约束（20 日均额 ≥ 5000 万 + 单票买入 ≤ 均日成交额 2%）天然限制容量。5 亿以上需放宽流动性门槛至 1 亿，可选标的池缩小约 30%。P3 优先级实现容量监控。 |
