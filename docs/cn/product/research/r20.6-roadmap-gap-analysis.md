## 调研范围

Web 搜索覆盖：聚宽/米筐/同花顺问财/通达信/Choice/FinGPT/FinRL 等平台 2025-2026 功能更新,AI 大模型在金融量化领域的新趋势,以及回测可视化行业最佳实践。

## 业界动态 (2025-2026) 摘要

| 平台 | 核心更新 | 我们是否有对应 |
|------|----------|---------------|
| **同花顺问财** | NL 选股已成标配,大模型深度整合,多模态交互 (文本+图表+语音),个性化推荐 | ❌ 无 NL 选股, 但 12 persona LLM 推理是差异化路径（见下文"不做的功能"） |
| **聚宽 JoinQuant** | GPT-4 驱动代码生成、AI 策略助手、另类数据源 (舆情/卫星)、优化回测引擎 | 部分：我们有 AI agent 回测但缺少另类数据源 |
| **米筐 RiceQuant** | 2025-2026 路线图：深度学习因子库、实时风控升级、AI 策略优化 | 部分：我们有因子 IC 分析但缺少深度学习因子挖掘 |
| **通达信** | AI 智能选股、ML 因子挖掘、Python API 开放 | 我们已超越（多策略融合 + LLM agent） |
| **FinRL** | 深度强化学习 (DRL) 交易框架，支持 DQN/DDPG/PPO/SAC | ❌ 未集成 DRL, 但可作为策略优化方向 |
| **FinGPT** | 金融 LLM 微调框架，金融情绪分析 | 我们已有等效实现 (news_sentiment agent + DeepSeek/Anthropic) |

## 行业趋势

1. **Agent 化交易系统** — 从单模型到多智能体协作（我们的 20-agent 架构已领先）
2. **端到端自动化** — 数据获取→策略研发→风控→执行全流程 AI 化（我们的 `--pipeline` 已实现）
3. **另类数据集成** — 舆情、卫星图像、链上数据（我们缺少此项，P2-9 宏观数据是第一步）
4. **深度学习因子挖掘** — 神经网络自动发现 Alpha 因子（我们有规则型因子 + IC 分析，缺少 DL 因子）
5. **可解释性 AI (XAI)** — 监管要求策略透明度（我们有因子瀑布 + `--explain`，行业领先）
6. **LLM 驱动 NL 查询** — 问财类自然语言选股成为标配（我们用 persona 推理差异化竞争）

## Features to Remove/Simplify（不做的功能）

| 功能 | 原因 |
|------|------|
| **NL 自然语言选股** | 问财已垄断零售市场，我们的 12 persona LLM 推理是差异化路径，不应正面竞争 |
| **港股通跨市场** | 不同数据源/监管/税制，超出当前范围 |
| **自定义因子 Python 编辑器** | 沙箱执行/版本控制/安全审计成本过高 |
| **移动端 App** | 当前 Web + CLI 已满足个人量化需求，移动 App 投入产出比低 |
| **深度学习因子挖掘** | 算力需求高，当前 4 策略因子 + IC 分析已覆盖主流需求，DL 因子属于研究方向而非产品功能 |

## 技术建议：P0-4 前端实现方案

基于业界最佳实践和我们的技术栈 (React + shadcn/ui + Tailwind)，建议：

**图表库选择**: Recharts (已有依赖) 或 Plotly.js (更强大但包体大)

**Dashboard 布局** (6 区域):
```
┌─────────────────────────────────────────────────────────┐
│  KPI Cards: Sharpe | Max DD | CAGR | Win Rate | Days    │
├───────────────────────────┬─────────────────────────────┤
│  Equity Curve             │  Drawdown (Underwater Plot)  │
│  (策略 vs 基准对比)        │  (水下图, 颜色编码严重度)     │
├───────────────────────────┼─────────────────────────────┤
│  Monthly Returns Heatmap  │  Rolling Returns            │
│  (year × month, RdYlGn)   │  (30d/90d/1y 三条线)        │
├───────────────────────────┴─────────────────────────────┤
│  Trade Log Table (可排序/筛选)                            │
└─────────────────────────────────────────────────────────┘
```

**关键交互**: 时间范围选择器 (1M/3M/6M/1Y/ALL)、hover tooltip 显示详细数值、策略 vs 基准切换、响应式布局。
